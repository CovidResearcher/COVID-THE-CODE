# COVID-THE-CODE
**Synthetic Molecular Evolution Thesis Model**

Pure Python implementation of the "MASTER EQUATION" for modeling SARS-CoV-2 evolution under NHC-TP (molnupiravir) pressure, ACE2 glycosylation remodeling, codon usage bias transformations, lung-to-brain tropism shift, and Warburg/HIF-1α immune exhaustion dynamics.
